# 1.动态规划

一般思路：
1.穷举法/暴力搜索
尝试写一个递归函数，完成穷举法的所有可能解

发现遍历中存在大量的重复计算，可以用哈希表将数据缓存下来
2.记忆化搜索/剪枝
遍历到相同的节点时，直接查表返回缓存的结果，避免重复计算

3.改写成迭代形式的动态规划算法
将计算过程表示出来，观察公式求解顺序，并尝试将递归形式改写成更简洁高效的迭代形式的动态规划算法。

```Python

问题；在一个无序数组，找出最长的递增的子序列的长度，nums=[10,9,2,5,3,7,101,18]

#暴力搜索时间复杂度O(2^n)*O(n)=(n*2^n)
def L(nums,i):
    """
    这个函数会返回从数组第i个数字开始的最长子序列长度
    """
    if i==len(nums)-1: #最后一个数字
        return 1

    max_len=1
    for j in range(i+1,len(nums)):
        if nums[j]>nums[i]:
            max_len=max( 1+L(nums,j),max_len)

    return max_len

def lengthOfLIS(nums):
    """
    这个函数会返回最长的递增子序列的长度
    """
    return max(L(nums,i) for i in range(len(nums)))

nums=[10,9,2,5,3,7,101,18]
print(lengthOfLIS(nums))


```

3.迭代/非递归的实现

```Python
#迭代时间复杂度O(n^2)
def lengthOfLIS(nums):
    """
    这个函数会返回最长的递增子序列的长度
    """
    n=len(nums)
    L=[1]*n

    for i in reversed(range(n)):
        for j in range(i+1,n):
            if nums[j]>nums[i]:
                L[i]=max(L[i],1+L[j])

    return max(L)


```

# 2.排序算法

```Python
def selection_sort(a,n):
    for i in range(1,n):
        ith=i
        for j in range(i+1,n+1):
            if a[j]<a[ith]:
                ith=j

        a[i],a[ith]=a[ith],a[i]
```


## 2.2冒泡排序
时间复杂度O(n^2)
```Python
def bubble_sort(a,n):
    for i in range(1,n):
        for j in range(1,n-i):
            if a[j-1]>a[j]:
                a[j-1],a[j]=a[j],a[j-1]
``` 


## 2.3插入排序
时间复杂度O(n^2)
```Python
def insertion_sort(a,n):
    for i in range(1,n):
        j=i
        while j>0 and a[j-1]>a[j]:
            a[j],a[j-1]=a[j-1],a[j]
            j-=1
```

时间复杂度O(n)

```Python
n = 0  # Array size
a = []  # Array (indexed from 1)
ps = []  # Prefix sum array


# Calculate prefix sum
def prefix_sum():
    global ps
    ps = a[:]
    # Or simply:
    # ps = list(itertools.accumulate(a))
    for i in range(1, n + 1):
        ps[i] += ps[i - 1]


# Query sum of elements in [l, r]
def query(l, r):
    return ps[r] - ps[l - 1]
```

```Python
n, m = 0, 0
a = []  # (n+1) x (m+1)
ps = []  # prefix sum array


# Calculate the prefix sum of 2D array.
def prefix_sum():
    global ps
    ps = [row[:] for row in a]  # Deep copy of a
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            ps[i][j] += ps[i - 1][j] + ps[i][j - 1] - ps[i - 1][j - 1]


# Find the sum of elements in submatrix [x1, y1] to [x2, y2].
def query(x1, y1, x2, y2):
    return ps[x2][y2] - ps[x1 - 1][y2] - ps[x2][y1 - 1] + ps[x1 - 1][y1 - 1]


if __name__ == "__main__":
```

# 4 二分
时间复杂度O(logn)

```Python
def binary_search(a,n,target):
    """
    这个函数会返回目标值在数组中的索引
    """
    left=0
    right=n-1
    while left<=right:
        mid=(left+right)//2
        if a[mid]==target:
            return mid
        elif a[mid]<target:
            left=mid+1
        else:
            right=mid-1
    return -1
   


```



# 5 字符串哈希
时间复杂度O(n)

**一.核心原理**
本质：将字符串映射为整数，用整数比较代替字符串比较，实现 O(1)​ 时间复杂度判断字符串相等。

哈希函数：f(string) = number，满足：

- 同一字符串 → 唯一哈希值（确定性）。

- 不同字符串 → 可能相同哈希值（哈希碰撞）。

目标：尽量减少碰撞，使“哈希相同 ≈ 字符串相同”。


**构造方法**
1.多项式哈希(最常用)：
将字符串视为P进制数

```
Python

hash(s) = (s[0]*P^(n-1) + s[1]*P^(n-2) + ... + s[n-1]*P^0) mod M
```

2.参数选择
|参数|建议|
|--|--|
|基数 P​|大于字符集的质数（如 131, 137, 91138233）|
|模数 M​|大质数（如 1e9+7, 1e9+9, 998244353）|
|字符映射​|非零值（a=1, b=2… 或 ASCII），避免映射为 0|
|溢出处理​|手动取模 或 使用 unsigned long long自然溢出|


3.前缀哈希预处理
```
Python
# 计算前缀哈希数组 h[]
h[i] = (h[i-1] * P + val(s[i])) % M
```
- 支持 O(1)​ 获取任意子串哈希值


**三.子串哈希提取**
区间哈希公式：
```
Python
hash(s[l:r+1]) = (h[r+1] - h[l] * P^(r-l+1)) mod M
```

- 需预计算P的幂数组powP[i] = P^i mod M，用于快速计算P^i mod M
- 适用于任意子串，无需重新计算哈希值，时间复杂度O(1)

**四.降低碰撞的策略**
1.**双模哈希**：同时使用两个不同的模数，两个哈希值都相等才认为字符串相同。

2.**随机化**：随机选择基数和模数（比赛时可固定常用值）。

3.**自然溢出**：依赖 unsigned long long自动对 2⁶⁴ 取模（简单但非质数）。


**五.典型应用**

**1.字符串匹配**

- 计算模式串哈希值。
- 滑动窗口计算文本串所有等长子串哈希值。
- 比较哈希值，统计出现次数。
- 时间复杂度：O(n+m)。

**2.最长回文子串**

- 预处理：在字符间插入特殊字符（如 #），统一奇偶长度。

- **方法一（二分）**：对每个中心二分长度，用子串哈希验证。复杂度 O(n log n)。

- **方法二（线性）**：维护当前最长回文半径，向右扩展。复杂度 O(n)。

- 需同时计算原串和反转串的哈希值。

**3.其他应用**

- 判断字符串是否循环同构。
- 快速比较两个字符串的子串是否相同。
- 检测字符串重复模式。

**六.注意事项**

- 减法取模后可能为负数，需调整为正数：(hash % M + M) % M。
- 字符映射值不能为 0，否则 "a"和 "aa"哈希值相同。
- 在严格要求无错的场景，建议配合其他算法（如 KMP）或双模哈希。




例子：
字符串：abcde   ，P=131 Mod=2^64

|字符|哈希值(映射)|Hash前缀值H|
|--|--|--|
|a|1|1（131*0+1）|
|b|2|133 （1* 131+2）即（（131*0+1）*131+2=（131*1）+2）|
|c|3|17426|
|d|4|2282810|
|e|5|299047115|


**求Hash('bcd')?**
=H[3]-[0] * P^3
=2282810-1 * 131^3
时间复杂度O(n)

## 6.2 最长公共子序列(LCS)
时间复杂度O(n^2)

1.a,b长度不一样
```Python

import sys

def main():
    data = sys.stdin.read().strip().split()
    it = iter(data)
    n = int(next(it))
    m = int(next(it))

    a = [0] * (n + 1)
    b = [0] * (m + 1)

    for i in range(1, n + 1):
        a[i] = int(next(it))
    for i in range(1, m + 1):
        b[i] = int(next(it))

    # dp[i][j] 表示 a 前 i 个和 b 前 j 个的最长公共子序列长度
    dp = [[0] * (m + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for j in range(1, m + 1):
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
            if a[i] == b[j]:
                dp[i][j] = max(dp[i][j], dp[i - 1][j - 1] + 1)

    print(dp[n][m])

if __name__ == "__main__":
    main()

```

2.a,b长度一样
```Python
import sys
import bisect

def main():
    data = sys.stdin.read().strip().split()
    it = iter(data)
    n = int(next(it))

    # a[x] 表示数字 x 在第一个排列中的位置
    a = [0] * (n + 1)
    for i in range(1, n + 1):
        x = int(next(it))
        a[x] = i

    # 读入第二个排列
    b = [0] * (n + 1)
    for i in range(1, n + 1):
        b[i] = int(next(it))

    # c[i] = b[i] 在 a 中的位置
    c = [0] * (n + 1)
    for i in range(1, n + 1):
        c[i] = a[b[i]]

    # f 数组用于 LIS（贪心 + 二分）
    f = [10**9] * (n + 1)
    f[0] = 0
    length = 0

    for i in range(1, n + 1):
        x = c[i]
        if x > f[length]:
            length += 1
            f[length] = x
        else:
            # 二分查找第一个 ≥ x 的位置
            pos = bisect.bisect_left(f, x, 1, length + 1)
            f[pos] = min(f[pos], x)

    print(length)

if __name__ == "__main__":
    main()


```
